Behind the scenes Meltano is tracking state using a State ID that's auto-generated based on the extractor name, loader name, and active environment name. To override the state and extract all data from the beginning again you can use the   --full-refresh   flag or the   MELTANO_RUN_FULL_REFRESH=1   environment variable.

If you'd like to view the state generated by the most recent run, you can use     meltano state get
